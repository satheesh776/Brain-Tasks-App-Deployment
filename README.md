# Brain Tasks App - Deployment Project
This project demonstrates deploying a React application to production using AWS services.

# Steps Completed
1. Cloned the application from GitHub.
2. Dockerized the application using a Dockerfile.
3. Created an ECR repository and pushed the Docker image.
4. Created an EKS cluster and deployed the application using:
   - deployment.yaml
   - service.yaml
5. Verified the application using the LoadBalancer URL.
6. Created CI/CD automation using AWS CodePipeline and CodeBuild.
7. Monitored logs using CloudWatch.

# Technologies Used
- React
- Docker
- AWS ECR
- AWS EKS (Kubernetes)
- CodePipeline
- CodeBuild
- CloudWatch
